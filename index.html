Matrix collection for Postman

Setup

  1. Download the collection
  2. Install Postman
  3. Create a workspace
  4. Click Import
  5. Select Matrix.postman_collection.json
  6. Create a global variable with name assetsURL and value https://twi1ightsparkle.github.io/matrix-postman/postman_assets/
  7. Create an environment with the following variables
    • url - this is the homeserver domain (the domain in your Matrix IDs)
    • delegatedURL - this is the domain where your homeserver is exposed
    • adminToken - type secret
    • userId
    • userToken - type secret
    • roomId
    • eventId
Customize the style and scripts

Download src/postman_assets/script.js or src/postman_assets/style.js from the GitHub repository, edit as required, then host them online. Next, update the URL in the global assetsURL variable.
